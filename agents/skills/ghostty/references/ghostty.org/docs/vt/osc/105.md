# Reset Special Colors (OSC 105)

Reset special colors.

1. 0x1B
   :   ESC
2. 0x5D
   :   ]
3. 0x31 0x30 0x35
   :   105
4. 0x3B
   :   ;
5. \_\_\_\_
   :   n
6. 0x1B
   :   ESC
7. 0x5C
   :   \

Reset the [special color](https://ghostty.org/docs/vt/concepts/colors.html#special-colors) at index `n`. `n` can be specified **zero or
more** times, separated with semicolons (`;`)
When `n` is not specified, all special colors are reset.
