# Ghostty Docs

Ghostty is a fast, feature-rich, and cross-platform terminal emulator
that uses platform-native UI and GPU acceleration.

## Get Started

Install Ghostty and run!
[Zero configuration required](https://ghostty.org/docs/config.html#zero-configuration-philosophy) to get up and running.

### Installation Instructions

Ready-to-run binaries for macOS. Packages or build from source for Linux.

* [Download](https://ghostty.org/download)
* [Build from Source](https://ghostty.org/docs/install/build.html)
