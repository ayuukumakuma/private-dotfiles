# Change Window Icon (OSC 1)

Change the name of the window icon.

Unimplemented

1. 0x1B
   :   ESC
2. 0x5D
   :   ]
3. 0x31
   :   1
4. 0x3B
   :   ;
5. \_\_\_\_
   :   t
6. 0x1B
   :   ESC
7. 0x5C
   :   \

Change the window icon name to `t`. The exact method of locating the icon
data through the icon name is platform-dependent.

Ghostty and most modern terminal emulators do not implement this sequence
as its behavior is not well-defined or even possible on all platforms,
which is also the conclusion reached by
[Foot](https://codeberg.org/dnkl/foot/pulls/1832#issuecomment-2304415).
