# Show Desktop Notification (OSC 9)

Show a desktop notification.

1. 0x1B
   :   ESC
2. 0x5D
   :   ]
3. 0x39
   :   9
4. 0x3B
   :   ;
5. \_\_\_\_
   :   t
6. 0x1B
   :   ESC
7. 0x5C
   :   \

Show a desktop notification with title `t`.

To avoid conflicting with the [ConEmu extensions](https://ghostty.org/docs/vt/osc/conemu.html), which also use OSC 9 due to
historical happenstance, the title should not begin with a number and then a
semicolon (`;`). In practice a collision of this kind should be rare.
Ghostty's parser will silently convert any invalid ConEmu sequence to a OSC 9
sequence, though this behavior should not be relied upon.

Originated from iTerm since before 2010.
