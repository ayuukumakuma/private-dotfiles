# Linefeed (LF)

Move the cursor down one line, scrolling if necessary.

1. 0x0A
   :   LF

This is an alias for [index (IND)](https://ghostty.org/docs/vt/esc/ind.html).

If [linefeed mode (mode 20)](https://ghostty.org/docs/vt/control/lf.html#TODO) is enabled, perform a
[carriage return](https://ghostty.org/docs/vt/control/cr.html) after the IND operation.
