# Backspace (BS)

Move the cursor backward one position.

1. 0x08
   :   BS

This sequence performs [cursor backward (CUB)](https://ghostty.org/docs/vt/csi/cub.html)
with `Pn = 1`. There is no additional or different behavior for
using `BS`.
