# Quick Start Guide

Rift is currently still in development and as such is not on any package managers.

## installation

Thus, the only way to install rift is via git:

```
git clone https://github.com/acsandmann/rift.git
```

once rift is installed you must build it (and even run it)using cargo (which obviously must be installed)

```
cargo run --bin rift --release
```

> in order for rift to begin managing windows it must first be turned on by pressing `Alt + Z` which is the default key combination for the `toggle_space_activated`

now to customize you must create your own config, this is thoroughly documented [here](https://github.com/acsandmann/rift/wiki/Config.html)
