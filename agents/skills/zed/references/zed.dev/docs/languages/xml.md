# [XML](https://zed.dev/docs/languages/xml.html#xml)

XML support is available through the [XML extension](https://github.com/sweetppro/zed-xml/).

* Tree-sitter: [tree-sitter-grammars/tree-sitter-xml](https://github.com/tree-sitter-grammars/tree-sitter-xml)

## [Configuration](https://zed.dev/docs/languages/xml.html#configuration)

If you have additional file extensions that are not being automatically recognized as XML just add them to [file\_types](https://zed.dev/docs/reference/all-settings.html#file-types) in your Zed settings:

```
  "file_types": {
    "XML": ["rdf", "gpx", "kml"]
  }
```
